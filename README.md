# haishin-plus

**haishin-plus** は、会場・配信・撮影・イベント運営・AIイベントHUBを組み合わせ、セミナー会場・貸し会議室・イベントスペースの空き時間を新しい収益源に変えるための共同事業型サービスです。

単なる配信代行ではなく、会場会社・配信会社・セミナー運営会社と連携し、「場所貸し」から「配信対応イベント運営パッケージ」への進化を目指します。

---

## Concept

> 空いている会場時間を、AIイベント運営事業に変える。

haishin-plus は、会場の空き枠、配信・撮影ニーズ、イベント運営の煩雑さをひとつの事業機会として捉えます。

これまで別々に手配されていた以下の要素を統合し、主催者・会場・配信チーム・運営スタッフがスムーズに連携できる仕組みを構築します。

- 会場予約
- 配信・撮影
- 機材・スタッフ手配
- 事前ヒアリング
- 当日進行管理
- 録画・アーカイブ納品
- 問い合わせ対応
- 請求・レポート
- AIによる運営補助

## Problems We Solve

会場ビジネスには、まだ収益化できていない空き時間があります。

代表的な課題は以下です。

- 平日昼間・夜間・短時間枠が埋まりにくい
- 会場・配信・撮影・受付・運営が別々に手配されている
- 配信対応が属人化しやすい
- 問い合わせ、見積、予約、当日運営、納品までがバラバラ
- 会場側が「場所貸し」から抜け出しにくい
- 配信会社が単発案件に依存しやすい
- 主催者がイベント準備に大きな負担を感じている

haishin-plus は、これらをまとめて商品化・標準化・システム化することを目指します。

---

## Solution

haishin-plus は、会場を **配信対応イベントパッケージ** として販売できる仕組みです。

単発の配信代行ではなく、以下をまとめて設計します。

- 配信パッケージ設計
- 価格設計
- 会場ごとの機材構成設計
- LP・営業資料作成
- 問い合わせ導線設計
- オペレーションマニュアル
- 配信スタッフ手配フロー
- 予約・見積・進行管理
- AIイベントHUB構築
- 共同ブランド展開

---

## haishin-plus HUB

haishin-plus HUB は、イベント運営の煩雑なやり取りを一元化するためのAIイベント運営HUBです。

想定機能は以下です。

- 予約管理
- 見積依頼
- 配信プラン選択
- 事前ヒアリング
- 機材・スタッフ手配
- 当日進行チェック
- 主催者とのチャット
- 会場スタッフとの連携
- 配信会社・撮影会社との連携
- 納品管理
- 請求・レポート管理
- AIによる問い合わせ対応
- AIによる進行補助
- AIによるタスク整理

目指す状態は、**「ログインして依頼すれば、イベント準備が進む」** ことです。

## Partnership Models

### 1. Venue Partner Model

会場会社が持つ空き枠・既存顧客・営業力を活用し、haishin-plus が配信商品設計・オペレーション設計・HUB構築を支援します。

### 2. Production Partner Model

配信・撮影会社の技術力を、単発受託ではなく、会場連携型の継続収益モデルへ展開します。

### 3. Seminar Operation Partner Model

セミナー運営会社が、会場・配信・受付・運営・録画納品までをワンパッケージで提供できるようにします。

### 4. Regional Expansion Partner Model

地方都市の会議室・ホール・ホテルなどと連携し、地域ごとに haishin-plus モデルを展開します。

## Design Direction

LPのデザイン方針です。

- BtoB向け
- 信頼感重視
- 未来感はあるが派手すぎない
- 会場運営会社の経営者・事業開発責任者が見て理解しやすい構成
- 配信機材、会場、スタッフ、管理画面、AI HUBがつながるビジュアル
- カラーはネイビー、ホワイト、ライトグレー、アクセントブルー系を想定
- SaaS感と現場感の両方を出す

## About IYASAKA

IYASAKA は、弱電工事、映像音響、ネットワーク、施設設備、ホテル・会場案件の現場経験を活かし、AIシステム開発・業務DX・自動化支援を行う会社です。

机上のDXではなく、現場で動く配線、機材、人、オペレーションを理解したうえで、AIとシステムを組み合わせることを重視しています。

haishin-plus では、配信事業を単なるWebサービスではなく、会場で本当に運用できる共同事業として設計します。

---

## Core Message

haishin-plus は、配信代行会社ではありません。

会場会社の空き時間を、配信・撮影・運営・AIイベントHUBによって共同で収益化する、イベントDX事業です.
